# ✨ TenseMaster - English Grammar Practice

An interactive web application designed to help English learners master past tense conjugations, including **Past Simple**, **Past Perfect**, and **Past Perfect Continuous**, along with **Question Tags**.

## 🚀 Live Demo
You can practice now at:  
**[https://past-tense-practice.vercel.app/](https://past-tense-practice.vercel.app/)**

---

## 📅 Project Information
- **Update Date**: 2026-03-29
- **Status**: ✅ Active / Question Bank Updated

## 🛠 Features
- **Dynamic Question IDs**: Every question across all categories has a unique identification number starting from 1000+, helping tutors and students identify specific exercises.
- **Smart Progress Tracking**: Real-time accuracy scoring and score tracking.
- **Tense Filtering**: Focus on specific tenses or take the "Global Challenge" with mixed questions.
- **Dark/Light Mode**: Aesthetic and comfortable practicing environment.
- **Expanded Pool**: Now supports up to **1,000 question** slots to prepare for large-scale practice banks.

## 📁 Technical Overview
- **Storage**: JSON-based static question bank for reliability and easy manual updates.
- **Build System**: Powered by **Vite** for fast, optimized delivery.
- **Styling**: Modern, responsive CSS with glassmorphism and subtle animations.
